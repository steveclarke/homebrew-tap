# Homebrew Tap

Personal Homebrew tap for Steve Clarke's tools.

## Installation

```bash
brew tap steveclarke/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `superthread` | CLI for Superthread project management |

## Installing a formula

```bash
brew install steveclarke/tap/superthread
# or after tapping:
brew install superthread
```
